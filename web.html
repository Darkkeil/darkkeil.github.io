Site web Sport Kho

Une autre partie de l'informatique non négligeable est la partie web. Nous avons créez un site web pour une association fictive nommée Sport Kho à partir d'une interview du demandeur du site. Nous avons mené toutes les étapes de la création d'un site, du maquettage sur Figma au codage en HTML et CSS du site sous VS CODE. Nous avons mis en place une charte graphique pour que toutes les pages de notre site soient uniformes ainsi que l'arborescence de notre site pour savoir quelle page mène vers quelle autre.
