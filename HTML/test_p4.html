Tests d'un puissance 4

Après la fin du codage de mon puissance 4 en langage C, il manquait une chose essentielle, effectuer des tests unitaires ! Après la fin de la phase de développement d'un logiciel, il faut tester son code avec tous les cas envisageables qui pourraient arriver lors de l'utilisation de l'application. L'étape du test permet de corriger les bugs de notre code ou de rajouter des cas de figure non prévus dans le code initial. C'est pour celà que j'ai testé mon puissance 4 dans toutes les situations qui peuvent arriver lors d'une partie pour m'assurer de son bon fonctionnement.
