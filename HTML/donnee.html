Visualisation de données

En base de données, pour peupler nos tables, il nous faut justement des données ! Nous pouvez les obtenir de deux manières : soit on insert des données brutes via la commande INSERT INTO , soit on la récupère en l'important depuis des fichiers csv dans les tables qu'on veut voir peuplée. Dans le cas de notre SAE, il nous fallait beaucoup de données pour peupler nos tables, nous avons donc utilisé un csv de données que nous avons importé dans notre schéma.
