Plateforme de Location Saisonnière

C'est dans le cadre de ma deuxième année de BUT informatique que j'ai réalisé à l'aide de mon équipe mon plus gros projet scolaire dans le développement d'application. L'objectif final était la réalisation d'une application web permettant la mise en ligne de logements fictifs et leur location par un propriétaire à un client sur l'ensemble du territoire breton. Après avoir pris connaissance du sujet et questionné nos professeurs sur les points dont nous n'étions pas sûr, nous avons commencé le code de cette application. Le langage utilisé pour le front de l'application a été le CSS via le framework Bootstrap. Pour le back nous avons utilisé du PHP natif ainsi que du MySQL et pour les interactions dynamiques : du Javascript. Comme le champ d'action de l'application couvrait uniquement la région bretonne naturellement nous avons décidé de nommer l'application Loc'Breizh.
