Synkronizator

Lors du cadre du projet où nous avons réalisé une plateforme de location saisonnière, j'ai réalisé une API en langage C permettant à un propriétaire de consulter ses logements et de consulter le planning de disponibilité de ces derniers. Chaque propriétaire à la possibilité de créer des clefs API dans un onglet spécifique et de se connecter via un terminal à l'API à l'aide de sa clef pour l'utiliser. L'API fonctionne sur le modèle client/serveur. Les utilisateurs par des commandes et options dans le terminal du Client requête le serveur afin de bénéficier de ses services.
